# 💳 CodeAlpha Credit Scoring Model

A Machine Learning project developed for the CodeAlpha Internship Program to predict whether a customer is eligible for a loan based on financial and personal details.

---

## 📌 Project Objective

The goal of this project is to analyze customer information and predict creditworthiness using a Machine Learning classification model.

This helps banks and financial institutions decide whether to approve or reject a loan application.

---

## 🚀 Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

---

## 🤖 Machine Learning Model

- Random Forest Classifier

The model was trained using historical customer loan data and tested for prediction accuracy.

---

## 📊 Features Used

- Age
- Annual Income
- Loan Amount
- Employment Length
- Credit History Length
- Previous Default History
- Home Ownership
- Loan Purpose
- Loan Percentage of Income
